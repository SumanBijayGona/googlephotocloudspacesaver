Free your Google storage

CloudCleaner downloads your Google Photos to your computer, renames them by date & location, and helps you reclaim cloud space — for free.

  • Saves photos as 2024-03-15_10-30-00_Austin-TX.jpg
  • Writes directly to a folder you pick on your hard drive
  • Tracks what's already saved — never downloads twice
  • Shows exactly how much cloud space you can recover
  • Runs entirely in your browser — nothing goes to any server

Requires Chrome or Edge 86+. Your photos go directly to your computer.

⚙️ One-time Google API setup

You need a free Google OAuth Client ID to access the Photos API. Takes ~10 minutes. See SETUP.md in the Android project for full instructions — the Google Cloud setup steps are identical.

Go to Google Cloud Console → Credentials → Create OAuth Client ID → type: Web application.
Add null (for opening from file) or http://localhost as an Authorized JavaScript origin.

ℹ️ One more step to free cloud space:
Your photos are now safely on your computer. Google doesn't allow third-party apps to delete photos directly — finish freeing space by doing this:
  1. Open photos.google.com
  2. Press Ctrl+A / Cmd+A to select all
  3. Trash icon → Move to Trash
  4. Trash → Empty Trash

Only do this after confirming all photos are in your chosen folder.
